# Pixel Craft

## Running server

Install modules

```
npm install
```

Start server

```
node index.js
```

## Compiling sass files

```
npm run sass
```

Requires that the sass ruby gem be install

## Environment variables

- IMAGE_STORE_URL must be set to couchdb url (with database name) i.e. http://local:host@localhost:5984/image_store
